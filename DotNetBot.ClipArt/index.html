.NET Bot Placeholder Images

This is a simple service that will deliver friendly images of the .NET Bot suitable for helping to design a web page or user interface. The background color of the image can be specified as yellow, cyan, blue, white, purple or none. If no color is specified, it will randomly choose one of the solid background colors.

Source code for this page is available at: https://github.com/csharpfritz/DotNetBot.ClipArt

You can request an image and specify the size and color of the image with a URL formatted as follows:

https://dotnetbotclipart.azurewebsites.net/300/400/yellow
