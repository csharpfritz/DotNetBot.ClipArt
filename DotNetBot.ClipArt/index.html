.NET Bot Placeholder Images

This is a simple service that will deliver friendly images of the .NET Bot suitable for helping to design a web page or user interface. The background color of the image can be specified as yellow, cyan, blue, white, or none.

Source code for this page is available at: https://github.com/csharpfritz/DotNetBot.ClipArt

You can request an image and specify the size and color of the image with a URL formatted as follows:

https://bot.dot.net/300/400/yellow
